# Fallout 76 Legendary Effects Tracker

Esta pwa tiene como objetivo permitirle a los jugadores de Fallout 76 realizar un seguimiento de cuales efectos
legendarios han desbloqueado hasta el momento, para que tengan a mano la información a la hora de desguazar sus ítems
y consideren si es mejor desguazarlos o canjearlos por vales legendarios

## Tecnologías utilizadas

- React (TypeScript)
- Tailwind CSS
- Motion
- React Icons
